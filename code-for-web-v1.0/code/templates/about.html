What we do?

The development team usually endeavors painstaking efforts to review and analyze many existing apps with similar purposes such as competitive analysis, design recommendation, code generation, and app testing. To assist different roles to do these tasks, in our prior work, we proposed StoryDroid and StoryDistiller to automatically generate the storyboards for Android apps with various types of features such as UI pages, UI components, layout code, and logic code. However, the main goal of these tools for generating storyboards is all aimed at exploring and parsing more app pages and lack some consideration of the presentation and interpretability of the results. To improve the usability and scalability, based these tools for the static and dynamic exploration of apps, and based on more effectively extracting and generating storyboards of Android applications with rich features, this paper presents an offline web-based tool, named StoryDroid+. StoryDroid+ provides an operation-friendly platform for users to explore apps and helps different stakeholders explore and understand applications from different perspectives through rich visual pages.
